# ECHO

A native iOS puzzle game: a turn-based black-and-white grid where you **fold** your moves into grey **echoes** and stack them to hold switches, block hazards, and reach the exit.

Built for its owner to play on his own iPhone — **offline, single-player, no App Store, no monetization, no network**. It is installed for **$0** by sideloading a free-signed build through [SideStore](https://sidestore.io), which re-signs the app with a free Apple ID and refreshes the 7-day certificate in the background.

## Build / run

Open `ECHO.xcodeproj` in **Xcode 27**, select your iPhone, and press **⌘R** (free signing → the install lasts 7 days). Run tests with **⌘U**. Requires iOS 17.0 or later.
