## Summary (plain language)
- **Works now:** Everything 1.03‚Äì1.06 shipped ‚Äî a state-driven board you move a black player around one tile at a time by swipe or adjacent-cell tap; off-grid/wall/closed-door moves are no-ops; a hidden shared turn counter; the **fold** mechanic that banks a run as a grey **echo** and rewinds to turn 0 with every echo replaying in lockstep; the rule that **touching a past self (or a hazard) dissolves present-you and restarts the current run** while folded echoes survive; per-room **walls / switches+doors / patrolling lethal hazards** all loaded from small JSON files in `Levels/` and all **derived per turn from positions**; **win** on reaching the exit alive (input locks); and an `echoBudget` cap on folds ‚Äî **plus the two everyday supporting controls (Phase 1.07):**
  - **Step back** (`GameState.stepBack()`): undoes one committed move of the current run. It pops the last `Direction` off `currentRun`, decrements the shared `turn` by one, and recomputes `player` by replaying the now-shorter run from `start`. It runs **no collision check and no win check** (it rolls back to a turn you already occupied alive) and **never touches `echoes`**. The whole derived world ‚Äî every echo, hazard, switch and door ‚Äî rolls back **for free** because each is a pure function of `turn` + positions (Observation re-reads them). It is a **no-op at turn 0** (nothing to undo) and is **refused while `hasWon`** (symmetry with `move`). Step-back is **intra-run only**: it never crosses the fold boundary and never un-folds a banked echo (D-028, D-030, D-031). There is **no redo** this phase (D-032).
  - **Reset run**: scraps the current attempt but **keeps every banked echo** ‚Äî wired directly to the existing `restartRun()` (player‚Üí`start`, `turn`‚Üí0, `currentRun` cleared, `hasWon` cleared, echoes preserved). It works mid-run and equally after a win. This is deliberately **distinct from `Clear`** (`clearEchoes()`, which also wipes the echoes and stays a debug-only op) (D-029).
- **Debug bar updated:** the throwaway strip in `ContentView` now reads **Fold / Step back / Reset run / Clear / Next**, plus the `turn N ¬∑ echoes M/budget B` readout and the **"Solved ‚úì"** stand-in. *Step back* is disabled at turn 0 (a one-line nicety; `stepBack()` is already a safe no-op). *Reset run* (keeps echoes) and *Clear* (wipes echoes) are separate buttons so the distinction is unmistakable. Still deliberately unstyled grey-box UI; the real in-room control layout and any backward-motion feel are Part 2/Part 3.
