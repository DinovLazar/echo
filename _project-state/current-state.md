It reads existing echo intent only — **no gameplay change** (D-051; handover §8/§6f).
  - **Guidance microcopy.** New `ECHO/Guidance/GuidanceController.swift` (`@MainActor @Observable`, Foundation-only) + `ECHO/Guidance/GuidanceOverlay.swift` (the SwiftUI view, mounted in `BoardView`'s board ZStack).
